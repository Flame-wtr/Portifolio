São eles:
  • Placa Arduino (qualquer modelo, como o Arduino Uno)
  • LEDs vermelho, amarelo e verde
  • Resistores de 220 ohms (ou próximos)
  • Protoboard
  • Fios de conexão (jumpers)

Aqui abaixo esta a modelo de como montar no arduino:

Ocodigo em blocos e escrito para funcionar o semáforo:

Codigo Escrito:

// C++ code // int counter; void setup() { pinMode(10, OUTPUT); pinMode(1, OUTPUT); pinMode(3, OUTPUT); pinMode(8, OUTPUT); pinMode(9, OUTPUT); pinMode(2, OUTPUT); for (counter = 0; counter < 10; ++counter) { digitalWrite(10, LOW); digitalWrite(1, LOW); digitalWrite(3, HIGH); delay(1000); // Wait for 1000 millisecond(s) digitalWrite(8, HIGH); delay(3000); // Wait for 3000 millisecond(s) digitalWrite(8, LOW); digitalWrite(10, LOW); digitalWrite(9, HIGH); delay(2000); // Wait for 2000 millisecond(s) digitalWrite(9, LOW); digitalWrite(10, HIGH); digitalWrite(3, LOW); delay(1000); // Wait for 1000 millisecond(s) digitalWrite(1, HIGH); delay(3000); // Wait for 3000 millisecond(s) digitalWrite(1, LOW); digitalWrite(2, HIGH); delay(2000); // Wait for 2000 millisecond(s) digitalWrite(2, LOW); digitalWrite(3, HIGH); } } void loop() { delay(10); // Delay a little bit to improve simulation performance }
